# Llama CLI Assistant

This project uses `llama-cpp` to provide a CLI interface for Llama models.

## Project Structure

- `README.md`: Documentation and examples.
- `replit.md`: Project overview and memory.

## Setup

The project depends on `llama-cpp` being available in the environment. It is installed via the Nix package manager.

## Workflow

A workflow is configured to run `llama-cli`. Note that you need to provide a model path.
